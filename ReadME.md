## Components 组件化

### 一、Components Vs Modules

1. 模块和组件均为可分离的、有独立功能的一种封装对象。
2. 模块强调功能性，其功能性并非一定与视图相关。一个完整的应用程序由多个模块组成。
3. 组件强调组合性，是一个视图片段的逻辑抽象。粒度比模块细，一个模块可以包含一个或多个组件。
4. 编程语言基元 --> 对象 --> 组件 --> 模块 --> 引用程序
